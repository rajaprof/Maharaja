## ğŸ§  About The Project

This repository contains a custom-built, highly interactive web portfolio and a print-ready digital resume. It is designed to showcase the intersection of **Data Engineering, Process Automation, and UI/UX Design**. 

The portfolio moves away from standard vertical scrolling, utilizing a custom horizontal scroll engine powered by GSAP to guide the viewer through a curated professional narrative.

## âœ¨ Key Features

* **GSAP Horizontal Scroll:** A bespoke horizontal scrolling experience linked to vertical mouse wheel movement.
* **Velocity Skew & Parallax:** Elements react to scroll speed, and CSS-rendered "Tech Mandalas" rotate in the background at varying depths.
* **Dynamic SVG Timeline:** A custom SVG line that draws itself as the user progresses through the story.
* **Print-Optimized HTML Resume:** A dedicated `resume.html` page that features a dark-mode screen UI, but automatically reformats into a clean, black-and-white, ATS-friendly document when printed (`Ctrl+P`).
* **Bento Grid Architecture:** A custom responsive CSS Grid layout for the art and UI gallery.
